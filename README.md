## ***Om meg*** 
*Etter flere år med modellering og analyser til eget bruk, bruker jeg nå denne nettsiden til å publisere og dele arbeidet mitt offentlig. Mitt fokus er å gå i dybden på kjente (og ukjente) selskaper og utfordre markedets konsensussyn.*


**Min analytiske tilnærming:**
*  **Bottom-up analyse:**  Modellene bygges konsekvent manuelt fra rådata i selskapenes egne års- og kvartalsrapporter.
*  **Fokus på verdiskapning:** Dypdykk i operasjonelle drivere, marginutvikling og kapitalavkastning
*  **Scenariobasert verdsettelse med sannsynlighetsvekting:** Kombinasjon av fundamentale DCF-modeller og relativ verdsettelse (multipler) fordelt på Bull/Base/Bear-case.
